# elephant_levels
Levels for a little game project

**Level structure**

TXT file 16x10, letters have special meanings and translates to tiles in game.

**Tiles**

* X: Empty space
* W: Wood block
* F: Fire block
* A: Water block
* B: Ball
* H: Hole
* E: Elephant (only 1 will be rendered)
* O: Food
* N: Enemy
* L: Laser
* D: Door

**Example**
```
WWWWWWWWWWWWWWWW
WXXXXXXXXXXXXXXW
WXXXXXXXXXXXXXXW
WXXXXXXXXXXXXXXW
WXEXXXXXXXXXOXXW
WXXXXXXXXXXXXXXW
WXXXXXXXXXXXXXXW
WXXXXXXXXXXXXXXW
WXXXXXXXXXXXXXXW
WWWWWWWWWWWWWWWW
```
